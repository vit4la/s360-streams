# Альтернативные способы получения токена VK

## Проблема: "Такой страницы нет" на oauth.vk.com

Это означает, что приложение не настроено для OAuth или URL неправильный.

## Решение 1: Проверить настройки приложения на vk.com/dev

1. Откройте https://vk.com/dev
2. Войдите в аккаунт
3. Перейдите в "Мои приложения"
4. Откройте ваше приложение (или создайте новое Standalone)
5. В левом меню откройте раздел **"Настройки"** (Settings)
6. Найдите раздел **"Базовый URL"** или **"Redirect URI"**
7. Добавьте: `https://oauth.vk.com/blank.html`
8. Сохраните изменения
9. Попробуйте снова получить токен

## Решение 2: Использовать другой redirect_uri

Попробуйте URL с другим redirect_uri:

```
https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://vk.com&scope=wall,groups&response_type=token&v=5.199
```

Или:

```
https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&scope=wall,groups&response_type=token&redirect_uri=https://oauth.vk.com/blank.html&v=5.199
```

## Решение 3: Получить токен через VK API напрямую (если есть доступ к приложению)

Если у вас есть доступ к настройкам приложения на vk.com/dev:

1. Откройте ваше приложение
2. Перейдите в раздел **"Настройки"** → **"Права доступа"** или **"Permissions"**
3. Убедитесь, что включены права: `wall`, `groups`
4. Сохраните изменения
5. Попробуйте снова получить токен через OAuth

## Решение 4: Использовать существующий токен из старого приложения

Если у вас было другое приложение, которое работало раньше:

1. Проверьте, есть ли у вас старый токен в логах или конфигах
2. Если токен истек - получите новый для того же приложения

## Решение 5: Создать новое приложение с нуля

1. На https://vk.com/dev создайте **новое** Standalone приложение
2. При создании укажите:
   - Тип: **Standalone приложение**
   - Название: любое
3. После создания сразу получите токен через OAuth

## Важно:

- Убедитесь, что используете правильный **Application ID** (не VK ID, а ID приложения с vk.com/dev)
- Приложение должно быть типа **"Standalone"**, не "Website" или "Mobile"
- Redirect URI должен быть настроен в настройках приложения
