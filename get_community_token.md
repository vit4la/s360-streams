# Как получить токен сообщества (Community Token) для закрытой группы

## Проблема
Токены формата `vk1.a.` могут не работать для закрытых групп. Для закрытых групп лучше использовать токен сообщества (Community Token).

## Решение: Получить токен сообщества через настройки группы

### Шаг 1: Откройте группу tennisprimesport

1. Откройте https://vk.com/tennisprimesport
2. Войдите в аккаунт, который является **администратором** или **редактором** группы

### Шаг 2: Перейдите в настройки группы

1. На странице группы нажмите **"Управление"** (или "Настройки")
2. В левом меню найдите раздел **"Работа с API"** или **"API"**
3. Если раздела нет - возможно, у вас нет прав администратора

### Шаг 3: Создайте ключ доступа сообщества

1. В разделе "Работа с API" нажмите **"Создать ключ доступа"** или **"Создать токен"**
2. Выберите права: `wall`, `groups`
3. Скопируйте токен сообщества

### Шаг 4: Обновите токен на сервере

```bash
nano /root/s360-streams/.env
# Замените VK_TOKEN= на токен сообщества
VK_TOKEN=токен_сообщества_здесь

systemctl restart vk-to-telegram.service
```

## Альтернатива: Если нет доступа к настройкам группы

Если вы не администратор группы, нужно:
1. Попросить администратора группы создать токен сообщества
2. Или использовать токен пользователя-участника группы (через OAuth)

## Важно:

- Токен сообщества работает для закрытых групп, если группа разрешила доступ через API
- Токен сообщества не привязан к IP адресу
- Токен сообщества обычно бессрочный
