# Инструкция по загрузке шаблона рамки из Figma

## Формат экспорта из Figma

1. **Размер шаблона:** 1320×1320 пикселей (квадрат)
2. **Формат:** PNG с прозрачностью (PNG-24)
3. **Структура шаблона:**
   - **Фон:** прозрачный (или темный, если нужен темный фон)
   - **Рамка:** лайм-зеленая (#7FFF00 или похожий цвет)
   - **Баннер в левом верхнем углу:** для текста заголовка (с диагональным срезом)
   - **Логотип в правом нижнем углу:** "SETKA^360" или логотип

## Как экспортировать из Figma

1. Выделите фрейм с шаблоном (1320×1320px)
2. В правой панели найдите раздел "Export"
3. Нажмите "+" для добавления экспорта
4. Выберите формат: **PNG**
5. Убедитесь, что включена опция **"Transparent background"** (прозрачный фон)
6. Нажмите "Export [название фрейма]"
7. Сохраните файл как `template.png`

## Куда загрузить файл

После экспорта из Figma:

1. Создайте директорию `assets` в корне проекта (если её нет):
   ```bash
   mkdir -p assets
   ```

2. Загрузите файл `template.png` в директорию `assets/`:
   ```bash
   # На сервере
   cd /root/s360-streams
   mkdir -p assets
   # Загрузите template.png в эту директорию
   ```

3. Структура должна быть:
   ```
   /root/s360-streams/
   ├── assets/
   │   ├── template.png    # Новый шаблон рамки
   │   └── logo.png        # Логотип (если нужен отдельно)
   ├── image_render_service.py
   └── ...
   ```

## Важные моменты

- **Прозрачность:** Область для фото должна быть прозрачной (или с полупрозрачным затемнением, если нужно)
- **Размер:** Точный размер 1320×1320px
- **Качество:** Экспортируйте в максимальном качестве (без сжатия)
- **Текст на шаблоне:** Если на шаблоне есть текст "Пример текста" - это будет заменено на реальный заголовок поста программно

## После загрузки

После загрузки `template.png` в `assets/`, код автоматически начнет использовать его вместо программной отрисовки.

Перезапустите сервис стилизации:
```bash
sudo systemctl restart image-render
```
