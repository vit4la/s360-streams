# Решение проблемы с закрытой группой VK

## Проблема
Группа `tennisprimesport` стала закрытой/приватной. Для закрытых групп нужен токен пользователя, который является **участником группы**.

## Решение 1: Получить токен от пользователя-участника группы

1. **Войдите в VK под аккаунтом, который является участником группы `tennisprimesport`**
   - Если вы не участник - подпишитесь на группу сначала

2. **Откройте ваше приложение на https://vk.com/dev**
   - Найдите ваше приложение (ID: 54378198 или другое)
   - Скопируйте Application ID

3. **Получите токен через OAuth:**
   - Откройте в браузере (замените YOUR_APP_ID на ID приложения):
   ```
   https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,groups&response_type=token&v=5.199
   ```
   - Нажмите "Разрешить"
   - Скопируйте `access_token` из URL

4. **Обновите токен на сервере:**
   ```bash
   nano /root/s360-streams/.env
   # Замените VK_TOKEN= на новый токен
   VK_TOKEN=ваш_новый_токен
   
   systemctl restart vk-to-telegram.service
   ```

## Решение 2: Дождаться, пока группа снова станет публичной

Если группа временно закрыта, можно просто подождать, пока она снова станет публичной.

## Решение 3: Использовать другую публичную группу

Если есть другая публичная группа с трансляциями тенниса, можно изменить `VK_GROUP_ID` в настройках.

## Важно:

- **Токен должен быть получен от пользователя, который является участником закрытой группы**
- Сервисные ключи VK ID не работают для закрытых групп
- Нужен именно токен пользователя через OAuth
