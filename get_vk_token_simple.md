# Как получить токен VK - простые способы

## Способ 1: Через браузер (самый простой, работает всегда)

1. **Откройте в браузере** (войдите в VK под аккаунтом администратора группы):
   ```
   https://oauth.vk.com/authorize?client_id=1&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,groups&response_type=token&v=5.199
   ```

2. **Нажмите "Разрешить"**

3. **Скопируйте токен из адресной строки**:
   - После перенаправления в адресной строке будет что-то вроде:
   ```
   https://oauth.vk.com/blank.html#access_token=ВАШ_ТОКЕН_ЗДЕСЬ&expires_in=86400&user_id=123456
   ```
   - Скопируйте часть между `access_token=` и `&` (это и есть ваш токен)

4. **Сохраните токен в .env файл на сервере**:
   ```bash
   nano /root/s360-streams/.env
   # Добавьте или замените строку:
   VK_TOKEN=ваш_токен_здесь
   ```

---

## Способ 2: Через настройки группы VK (если вы администратор)

1. Откройте группу `tennisprimesport` в VK
2. Перейдите: **Управление** → **Работа с API**
3. Создайте ключ доступа с правами: `wall`, `groups`
4. Скопируйте токен

---

## Способ 3: Через VK Dev (https://vk.com/dev)

1. Перейдите на https://vk.com/dev
2. Создайте **Standalone-приложение**
3. Получите `access_token` с правами: `wall`, `groups`
4. **ВАЖНО**: Токен должен быть получен от пользователя, который является администратором группы

---

## Способ 4: Через мобильное приложение VK (если есть)

Некоторые версии мобильного приложения VK позволяют получить токен через настройки разработчика, но это редко используется.

---

## После получения токена:

```bash
# На сервере
cd /root/s360-streams
nano .env
# Добавьте: VK_TOKEN=ваш_новый_токен

# Перезапустите сервис
systemctl restart vk-to-telegram.service

# Проверьте логи
tail -n 20 /root/s360-streams/vk_to_telegram.log
```

---

## Важно:

- Токен должен быть получен от **администратора группы** `tennisprimesport`
- Если группа стала приватной, токен обычного пользователя не сработает
- Токены обычно действуют 24 часа, но можно получить бессрочный через настройки приложения

---

## Если ничего не помогло:

Проверьте, не стала ли группа приватной:
1. Откройте https://vk.com/tennisprimesport
2. Если группа закрыта - нужно либо сделать её публичной, либо использовать токен администратора
